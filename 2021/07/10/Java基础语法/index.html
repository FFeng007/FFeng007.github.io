二、标识符、关键词

1.标识符命名规则

  • 所有的标识符都应该以字母、美元符号、下划线开始
  • 首字符之后可以是字母、美元符号、下划线、数字的任意组合
  • 不能用关键词命名
  • 大小写敏感

2.关键词

三、数据类型

1.八大基本数据类型

  • 字节型byte 取值范围：0 - 255 占1个字节

  • 短整型short 取值范围：-32,768 ~ 3276 占2个字节

  • 整形int 取值范围：-2,147,483,648 ~ 2,147,483,647 占4个字节

  • 长整型long 取值范围：
    -9223372036854775808~9223372036854775807 占8个字节

  • 单精度浮点型float占4个字节

    负数范围:-3.402823E38 ~ -1.401298E-45
    正数范围:1.401298E-45 ~ 3.402823E38

  • 双精度浮点型double 占8个字节

    负数范围:-1.797,693,134,862,32E308 ~-4.940,656,458,412,47E-324
    正数范围:4.940,656,458,412,47E-324 ~1.797,693,134,862,32E308

  • 字符型char 用单引号括起来 占2个字节

  • 布尔型boolean （ true false） 占1个字节

2.引用数据类型

类（String类）、接口、数组

引用数据类型的大小统一设置为4个字节，记录的是其引用对象的地址。

3.数据类型扩展

  • 整型扩展：进制表示：二进制 0b 开头；八进制 0 开头；十进制；十六进制 0x 开头；

  • 浮点型扩展：由于字长有限，浮点数能够表示的数是有限的、离散的，一般存在舍入误差，很多数字无法完全精确表示，结果只能接近但不等于。所以避免在比较中使用浮点。（浮点类型不适合在不容许误差的金融计算领域，若要进行精确数字计算要使用BigDecimal类）

  • 字符型扩展：”A”是字符串，’A’是字符，

  • 布尔型扩展

四、类型转换

由于Java语言是强类型语言，所以在进行有些运算的时候要进行类型转换。

1.自动类型转换

容量小的可以自动转换为容量大的数据类型。

从小到大：byte,short,char < int < long < float < double

2.强制类型转换

容量大的数据类型需要 强制转换 成容量小的的数据类型。 （type）var

3.常见问题

  • 溢出问题

  • L和l问题

    由于小写L和数字1很相似，为了更方便的识别代码，不要命名名字为小写l的变量，long类型使用大写L不使用小写l

五、变量和常量

1.变量

  • 变量：可以变化的量

  • 变量作用域（变量分为三种）

    • 类变量（静态变量）
  • 实例变量（成员变量）
  • 局部变量

2.常量

  • 常量：值被设定后，在程序运行过程中不能被改变。
    final 常量名 = 值;

3.命名规则

  • 类名：首字母大写和驼峰原则 Demo DemoScore
  • 类成员变量、局部变量和方法名：首字母小写和驼峰原则 demo demoScore
  • 常量：大写字母和下划线 DEMO_SOCRE

六、运算符

1.基本运算符（二元运算符）

  • 加减乘除（+ - * /） 取模（%）
  • 整数运算：俩个操作数，一个为Long，结果为Long；没有Long时，结果为int；即使操作数全为byte或者short，结果也是int。
  • 浮点运算：俩个操作数，一个为double，结果为double；都为float才为float。
  • 取模运算：负数%负数＝负数； 负数%正数＝负数； 正数%负数＝正数；
  • 关系运算符 (== != > < <= >=) 返回布尔值

2.一元运算符

  • 自增++ 自减–
  • Math类

3.条件运算符（三元运算符）

a ? b : c 如果a是true就是b，否则就是c

4.逻辑运算符

  • 与&& （and） 或||（or） 非（取反）

  • 短路运算：逻辑与只要有一个为false, 则直接返回false. 逻辑或只要有一个为true, 则直接返回true;

5.位运算符

  • 位运算符应用于整数类型(int)，长整型(long)，短整型(short)，字符型(char)，和字节型 (byte)等类型

6.扩展赋值运算符

  • a += b;//a = a + b
    a -= b;//a = a - b
    a /= b;//a = a / b
    a *= b;//a = a * b

七、包

  • 包的作用：把功能相似或相关的类或接口组织在同一个包中，方便类的查找和使用；同一个包中的类名字是不同的，不同的包中的类 的名字是可以相同的。

  • 包的创建：一般用公司域名倒置， www.baidu.com 包名 com.baidu.www

  • 包的导入：为了可以使用某一包中的成员，需要用import导入包。通配符*可以导入包下的所有类

八、JavaDoc

  1. JavaDoc是可以将注释生成HTML文档的的技术。

    @author 作者名

    @version 版本号

    @since 指明需要最早使用的jdk版本

    @param 参数名

    @return 返回值情况

    @throws 异常抛出情况

  1. 用命令行生成JavaDoc文档：javadoc -encoding UTF-8 -charset UTF-8 java文件

一.Spring Security

  • 权限框架,为java应用程序提供身份验证和授权
  • 用户认证 Authentication
    • 验证某个账户是否为系统中的合法主体
  • 用户授权 Authorization
    • 验证某个用户是否有权限执行某个操作

二.练习

1.认识Spring Security

  • Spring默认的安全框架,导入spring-boot-starter-security模块,进行简单配置,即可实现安全管理
    • WebSecurityConfigurerAdapter： 自定义Security策略
    • AuthenticationManagerBuilder：自定义认证策略
    • @EnableWebSecurity：开启WebSecurity模式
  • 认证 Authentication
    • 通过用户名,ID,密码等验证身份,进行认证
  • 授权 Authorization
    • 在认证后,授予该账户相应的权限

outline：绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用。

盒子模型

  • width 内容宽度
  • height 内容高度
  • border 边框
  • margin 外边距 透明的
  • padding 内边距
    • box-sizing: border-box：通过压缩内容实现固定盒子大小，（内容+内边距+边框=宽高）

浮动

float：更改⻚⾯元素的排列布局⽅式，能实现我们⻚⾯的布局效果。

  • left 左浮动
  • right 右浮动
  • none 默认值 不浮动

定位

  • top、right、bottom、left ==> 距离多少

相对定位：realtive ==> 相对于自身

  • 以自己为参照物，移动位置，原来的位置信息去不掉
  • 不脱离文档流，文字不会顶上去
  • 相对自身进行位置偏移，元素在原文档的位置继续保留

绝对定位：absolute ==> 相对于父级空间

  • 以父级为参照物，没有父级则以body为参照物
  • 脱离文档流，文字会顶上去
  • 相对有定位方式的父元素，若父元素没设置定位，找body控件为参照物

固定定位：fixed ==> 以浏览器为参照物

层叠：z-index ==> 层叠模式

  • 层叠模式，数字大的在上边

字符函数

  • substr(x,y) ==> 截取内容，x起始位置，y长度
  • substring(x,y) ==> 截取内容，x起始位置，y结束位置 ，左闭右开
  • charAt(x) ==> x索引位置的值
  • indexOf(x,y) ==> 获取x的索引值，y是开始检索的位置
  • split(“x”) ==> 以x分割
  • concat(“x”,”y”,”z”) ==> 拼接xyz 数组拼接会放在新的数组中
  • replace(“x”,”y”) ==> 将x替换成y

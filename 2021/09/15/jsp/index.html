一、JSP

  1. JSP：java server page Java服务器页面

  2. 特点：java代码和HTML分离，降低开发难度，使用简单。本质就是servlet

1.JSP基本语法

  • <%%> jsp代码块：嵌入Java代码<% java 代码块%>
  • <%!%> jsp声明构造快：声明变量和方法<%! int a = 10 %>
  • <%=%> jsp输出指令：显示java代码的执行结果<%=a%>

2.转移过程

  • HTML代码，就会被转换为：out.write(“\r\n”);

3.九大内置对象

  • PageContext 页面上下文对象，存东西
  • Request 请求，存东西
  • Response 响应
  • Session 用户会话，存东西
  • Application ServletContext 应用全局对象，存东西
  • config ServletConfig 应用配置对象
  • out PrintWriter 输出对象
  • page 当前页面对象，了解
  • exception 应用异常对象
